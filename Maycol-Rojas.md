## Desarrollo de Historias de Usuario y Suites de Pruebas - Maycol Rojas Velásquez

### Historias de Usuario (US)

A lo largo del curso, Maycol Rojas Velásquez ha participado activamente en la definición y desarrollo de diversas historias de usuario que forman parte integral del proyecto. Las historias de usuario están diseñadas para capturar los requisitos funcionales desde la perspectiva del usuario final. Estas historias se centran en las necesidades y expectativas del usuario, proporcionando un marco claro para el desarrollo iterativo del software.

### Technicals Stories (TS)

Maycol Rojas Velásquez ha realizado contribuciones significativas en la creación y ejecución de suites de pruebas que aseguran la calidad del software desarrollado. Las suites de pruebas abarcan una variedad de casos de prueba diseñados para validar el cumplimiento de los requisitos funcionales y no funcionales establecidos en las historias de usuario. Estas pruebas incluyen escenarios positivos y negativos para garantizar que el sistema funcione correctamente bajo diferentes condiciones.
